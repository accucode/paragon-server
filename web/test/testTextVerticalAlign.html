Test vertical text centering. What we want is some text vertically centered on the left, with an image on the right. We want to avoid relying on fixed size layouts.

Simply applying vertical-align:middle to the outer div and/or inner span doesn't work.

some text

Here we wrap the text and image in divs with a display:table-cell. This tell the browser to use the layout rules that are traditionally used for TDs. Now we can apply the vertical-align:middle to the mock cell and the nested inline text does center vertically.

some text

Here we use the same display:table-cell trick, but with lines of text on the right instead of an image. Note that instead of float:right we set the first cell's size to 100%; this means that we need to set white-space:nowrap on the righthand cell to keep the text from getting compressed.
